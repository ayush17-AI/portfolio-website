# Ayush Naraniwal Portfolio

A minimalist, high-performance professional portfolio website built with modern web technologies. This project focuses on clean typography, premium animations, and a seamless user experience.

## 🚀 Key Features
- **Minimalist Design**: A distraction-free landing page with a primary focus on content and vision.
- **Premium Animations**: Smooth entrance and scroll-triggered effects using **Framer Motion**.
- **Interactive 3D Background**: Subtle 3D floating shapes powered by **React Three Fiber**.
- **Performance Optimized**: Built on **Next.js 14** for fast loading and optimal stability.
- **Clean Aesthetic**: A sophisticated white and warm brown theme with refined spacing.

## 🛠️ Tech Stack
- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **3D Graphics**: [React Three Fiber](https://docs.pmnd.rs/react-three-fiber/) / [Three.js](https://threejs.org/)
- **Icons**: [Lucide React](https://lucide.dev/)

## 💻 Getting Started

### Prerequisites
- Node.js (Latest LTS version recommended)
- npm or yarn

### Local Development
1. **Clone the repository:**
   ```bash
   git clone https://github.com/ayush17-AI/portfolio-website.git
   cd portfolio-website
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **View the site:**
   Open https://portfolio-website-three-azure-68.vercel.app/ in your browser.

## 📦 Deployment

This project is ready to be deployed on platforms like **Vercel** or **Netlify**:
1. Connect your GitHub repository to Vercel.
2. The build settings should automatically be detected (Next.js).
3. Click **Deploy**.
